# Langelihle Natasha Kumbemba - Portfolio

A beautiful, animated portfolio website showcasing the modeling work and personality of Langelihle Natasha Kumbemba.

## Features

- ✨ Animated hearts and roses floating across the page
- 📸 Gallery with 10 stunning photos
- 🎬 Automatic slideshow of featured images
- 💖 Beautiful hover effects and transitions
- 📱 Fully responsive design
- 🎵 Favorite song and Bible verse sections

## Local Development

```bash
npm install
npm run dev
```

Visit `http://localhost:5173` to view the site.

## Build for Production

```bash
npm run build
```

## Deploy to Netlify

1. Push your code to GitHub
2. Go to [Netlify](https://netlify.com)
3. Click "Add new site" → "Import an existing project"
4. Connect your GitHub repository
5. Netlify will read build settings from the repo-root `netlify.toml`
6. Click "Deploy site"

## Deploy to GitHub Pages

1. Push your code to GitHub
2. Go to your repository Settings → Pages
3. Under "Build and deployment", select "GitHub Actions"
4. The repo-root workflow `.github/workflows/deploy.yml` will deploy on every push to `main`

## Manual Deployment

Build the project and upload the `dist` folder to any static hosting service.
